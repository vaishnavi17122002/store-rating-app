# ⭐ Store Rating App

A Full Stack Store Rating Application developed using React.js, Node.js, Express.js, and MySQL.

The application allows users to register, login, rate stores, and manage ratings based on different user roles.

---

# 🚀 Features

## 👨‍💼 Admin

- Add Stores
- Add Users
- Manage Store Owners
- Dashboard Analytics
- View All Users
- View All Stores
- Filter & Search Users
- Filter & Search Stores
- View Ratings

---

## 👤 Normal User

- Register/Login
- Search Stores
- Submit Ratings
- Update Ratings
- View Store Ratings
- Update Password

---

## 🏪 Store Owner

- Login
- View Users Who Rated Store
- View Average Store Rating
- Update Password

---

# 🔐 Authentication

- JWT Authentication
- Role-Based Access Control
- Protected Routes

---

# ✅ Form Validations

## Name
- Minimum 20 Characters
- Maximum 60 Characters

## Password
- 8–16 Characters
- At Least One Uppercase Letter
- At Least One Special Character

## Email
- Proper Email Validation

## Address
- Maximum 400 Characters
